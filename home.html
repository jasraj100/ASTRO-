SQL Injection ‚Äî Login Parameter
CRITICAL
CVE: N/A (Custom)  |  CVSS: 9.8
Description: The login endpoint is vulnerable to time-based blind SQL injection via the username parameter, allowing unauthenticated access to the full database.
Remediation: Implement parameterized queries, input validation, and WAF rules.
Weak SSH Configuration (Root Login Enabled)
HIGH
CVE: CVE-2023-38408  |  CVSS: 7.5
Description: SSH daemon allows root login and uses weak cipher suites, exposing the server to brute force and man-in-the-middle attacks.
Remediation: Disable root SSH login, enforce key-based auth, update cipher list.
Missing Security Headers (X-Frame-Options, CSP)
MEDIUM
CVE: N/A  |  CVSS: 4.3
Description: The web application does not implement recommended HTTP security headers, leaving it vulnerable to clickjacking and XSS attacks.
Remediation: Add X-Frame-Options: DENY, Content-Security-Policy, X-Content-Type-Options headers.
